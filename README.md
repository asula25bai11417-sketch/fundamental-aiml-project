#  Movie Recommendation System

##  Overview
This project is a simple movie recommendation system that suggests similar movies based on a given movie using machine learning.

##  Features
- Recommends top 5 similar movies
- Uses movie overview for similarity
- Easy to run and understand

##  Technologies Used
- Python
- Pandas
- Scikit-learn

##  Dataset
- TMDB Movies Dataset
- TMDB Credits Dataset

##  Installation

1. Clone the repository:
git clone https://github.com/asula25bai11417-sketch/movie-recommender.git

2. Install dependencies:
pip install -r requirements.txt

3. Run the project:
python recommender.py

##  Usage
Enter a movie name when prompted and get recommendations.

##  Example
Input:
Avatar

Output:
- Guardians of the Galaxy
- John Carter
- Star Trek
- The Fifth Element
- Interstellar
